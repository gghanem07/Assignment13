
In this assignment, I worked on integrating and debugging the system to ensure everything worked end-to-end. I worked on connecting the JWT authentication routes with the front-end login and registration pages, making sure user input was properly validated and handled on both the client and server sides.
One of the main challenges I encountered was debugging issues between the frontend and backend. For example, I had to fix how error messages were displayed when the backend returned structured validation errors. I also ran into several environment related problems, including database connection issues, port conflicts with a local PostgreSQL instance, and missing database tables. Resolving these helped me better understand how Docker containers, databases, and application startup processes interact.
Another important part of this assignment was implementing Playwright E2E tests. I created tests that simulate real user behavior, such as registering, logging in, and handling invalid inputs. I also updated the CI/CD pipeline to ensure these tests run automatically, which focuses the importance of automated testing in maintaining application reliability.Overall, this assignment helped me gain practical experience troubleshooting full-stack issues and ensuring that different parts of an application work together correctly.
